# DEFNet
DEFNet: Dual-Branch Enhanced Feature Fusion Network for RGB-T Crowd Counting（[paper](https://ieeexplore.ieee.org/document/9889192)）


# Requirements
Python3.6, Pytorch 1.5.0+, Cuda 10.2, Opencv-python

# Dataset
